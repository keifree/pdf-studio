# Antigravity PDF Studio 📖 (Google Drive & 右綴じ対応PDFリーダー)

Adobe Acrobat Readerの代替を目指し構築された、高機能・モダンデザインのWeb PDF閲覧・注釈アプリケーションです。

---

## 🌟 主な機能と特長

### 1. ☁️ Google Drive 完全連携（ローカル保存不要の直接ストリーム閲覧・更新）
- **ローカルダウンロード不要**: Google Drive上のPDFをブラウザのRAMメモリ（`ArrayBuffer`）に直接読み込んで高速レンダリング。
- **ダイレクト上書き保存**: アノテーション（手書き・ハイライト・コメント）や右綴じ属性を適用したPDFを、Google Drive上の元のファイルへ直接更新（`PATCH`/`multipart`）可能。

### 2. 📖 右綴じ（RTL: Right-to-Left）本属性の付与 ＆ 見開き表示
- **右綴じ（漫画・和書・縦書き書籍）完全対応**: 2ページ表示（見開き）にした際、画面右側が奇数ページ（表紙側）、左側が偶数ページ（進む側）になる日本の書籍標準の右綴じ表示をワンタップで切替。
- **キーボードナビゲーション連動**: 右綴じモード時は、`←` キーでページが進み、`→` キーで戻る標準漫画ビューアの直感的な操作を実現。
- **PDFメタデータ書き出し**: `pdf-lib`を用いて、PDF仕様のカタログオブジェクト（`/ViewerPreferences << /Direction /R2L >>`）を埋め込んでエクスポート。対応するPDF閲覧ソフトで開いた際にも自動的に右綴じで開かれます。

### 3. 🔍 高機能な表示サイズ調整（ズーム）
- **幅に合わせる (Fit Width)**: ウィンドウ幅に最適化。
- **全体表示 / 高さ (Fit Height)**: 1ページ全体を収める表示。
- **100% (原寸表示)**: 原寸大表示。
- **50%〜200% カスタムズーム**: ドロップダウンで瞬時に拡大・縮小。

### 4. ✏️ 豊富なアノテーション機能
- **手書きペン (Freehand Ink)**: 筆圧・太さ・カラー選択・消しゴム対応。
- **蛍光ペンハイライト (Highlighter)**: テキストや図表の上から引ける半透明ハイライト。
- **ピン留めコメント (Sticky Notes)**: PDF上の任意の位置をタップして吹き出し注釈ピンを設置。右サイドバーでコメントスレッド（投稿者・時刻・本文）を一元管理。

## 🚀 起動方法

### 方法1: Pythonローカルサーバーで起動（推奨）
コマンドプロンプトやTerminalで以下を実行します：

```bash

python server.py
```

`index.html` ダブルクリックでもサンプルPDFの動作確認やローカルPDFの編集が可能です。

## 🔑 Google Drive OAuth 2.0 設定手順

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、プロジェクトを作成します。
2. **Google Drive API** および **Google Picker API** を有効化します。
3. **OAuth 同意画面** を設定し、スコープに `https://www.googleapis.com/auth/drive.file` を追加します。
4. **認証情報** > **OAuth 2.0 クライアント ID**（ウェブ アプリケーション）を作成し、承認済みの JavaScript 生成元に `http://localhost:8080` を追加します。
5. 本アプリのヘッダーにある「Google Drive」ボタンをクリックし、発行された **Client ID** を入力・保存してください。
